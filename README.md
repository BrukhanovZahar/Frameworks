## Датасеты

### Классификация: Credit Card Fraud Detection
- **Источник**: [Kaggle - Credit Card Fraud Detection](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)
- **Задача**: Обнаружение мошеннических транзакций по кредитным картам
- **Практическое применение**: Финансовая безопасность, предотвращение мошенничества

### Регрессия: Student Performance Dataset
- **Источник**: [Kaggle - Student Performance Dataset](https://www.kaggle.com/datasets/larsen0966/student-performance-data-set)
- **Задача**: Предсказание итоговой оценки студента (G3) на основе различных факторов
- **Практическое применение**: Образовательная аналитика, раннее выявление студентов группы риска
